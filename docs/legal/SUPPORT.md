**Email:** support@msgguard.app  
**Web:** https://msgguard.app/support

## Enable Message Filter Extension

1. Settings → Messages → Unknown & Spam
2. Select **MsgGuard** → enable
3. Open MsgGuard app → Settings → confirm extension toggle

## FAQ

### SMS privacy
Filtering runs on-device by default. Cloud AI requires Pro + explicit opt-in in Settings.

### Restore subscription
Settings → Restore Purchases, or App Store account subscriptions page.

### Report misclassification
Use **Samples** to label locally, or **Feedback** with Trace ID.

## Enterprise / API

Admin API requires Bearer token. Enterprise deployments may require mTLS for `/api/v1/admin/*` — see Cookbook 33.

Enable Message Filter

  1. Open Settings → Messages → Unknown & Spam
  2. Select MsgGuard and turn on filtering
  3. In the MsgGuard app, confirm Extension is enabled under Settings

FAQ

Does MsgGuard upload my SMS?
No, by default. Classification runs on your device. Cloud AI (Pro, opt-in) sends single messages to our API only when enabled.

How do I restore Pro?
Settings → Subscription → Restore Purchases, or App Store → Account → Subscriptions.

False positive / missed spam?
Use the in-app Samples tab to label messages and retrain the local model, or submit Feedback with Trace ID from Settings.

Contact

Email support@msgguard.app with iOS version and Trace ID if available.
